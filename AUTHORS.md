## How to cite

See [CITATION.cff](CITATION.cff) for machine-readable citation metadata.

For software citation:

```bibtex
@software{chromagraphnet2026,
  author       = {Roy and Pavel, Juboraj Roy},
  title        = {ChromaGraphNet: multi-modal graph-attention prediction
                   of 3D genome architecture},
  year         = {2026},
  publisher    = {GitHub},
  url          = {https://github.com/USERNAME/chromagraphnet},
  version      = {0.1.0}
}
```
